# Python 从入门到实践（第2版） - 项目2：数据可视化 练习代码

## 项目简介
本仓库包含《Python 从入门到实践（第2版）》中 **项目2：数据可视化** 的练习代码。通过 Matplotlib 和 Plotly 等库，学习如何用 Python 生成各种图表，包括折线图、柱状图、散点图等。

## 内容概览
- **基础图表**：折线图、柱状图、散点图
- **数据清洗与可视化**：处理 CSV/JSON 数据并绘制图表
- **高级应用**：调用API访问网站并获得数据并进行数据可视化
